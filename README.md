# Todo List App

A simple Todo List application built with HTML, CSS, and JavaScript.

## Features
* Add new tasks
* Delete existing tasks
* Responsive design
* Clean and simple UI

## Technologies Used
* HTML5
* CSS3
* JavaScript (ES6+)

## How to Use
1. Clone the repository to your local machine
2. Open the `index.html` file in your browser
3. Add new tasks by typing in the input field and clicking the "Add Task" button
4. Delete tasks by clicking the "Delete" button next to each task

## Code Structure
* `index.html`: The main HTML file for the application
* `style.css`: The CSS file for styling the application (included in the HTML file)
